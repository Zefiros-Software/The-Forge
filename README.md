# Json
To use [json](https://github.com/nlohmann/json) in [ZPM](http://zpm.zefiros.eu), just use:

# [ZPM](http://zpm.zefiros.eu) Installation
In `.package.yml`
```yaml
libraries:
  - name: Zefiros-Software/Json
    version: '^3.0.0'
```

In `zpm.lua`
```lua
zpm.uses "Zefiros-Software/Json"
```
